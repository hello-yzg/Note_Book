盛最多水的容器

C++ Tips

  1. vector 长度： *.size()
  2. 数组长度：std : size(*)

Java Tips

  1. 数组长度：*.length

Python Tips

  1. Python的设计哲学之一是”显式优于隐式”，因此不能使用 ++ 或 – 运算符，而是使用 += 1 或 -= 1 实现自加或自减。

题目：

给定一个长度为 n 的整数数组 height 。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i]) 。

找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

说明：你不能倾斜容器。

贪心双指针

时间复杂度：O(n)

空间复杂度：O(1)
